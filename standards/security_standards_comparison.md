# macOS Security Standards Comparison

Comparison of rules across **CIS Level 2**, **CIS Controls v8**, and **NIST 800-53r5 High** baselines.

## Coverage Summary

| Standard | Rules |
|---|---|
| CIS L2 | 117 |
| CIS v8 | 175 |
| 800-53r5 High | 218 |
| Total unique | 266 |

- Rules in all 3 standards: **74**
- Rules in exactly 2 standards: **96**
- Rules in exactly 1 standard: **96**
